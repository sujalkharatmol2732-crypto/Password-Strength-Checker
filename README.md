# Password Strength Checker

A graphical Python application built with Tkinter that evaluates password strength using visual circular indicators, provides strength suggestions, and supports visible password input for security testing and training.

## Features
- Real-time password strength analysis with color-coded circular meter (green = strong, yellow = medium, red = weak).
- Custom scoring based on length, uppercase/lowercase letters, numbers, and special characters.
- Password visibility toggle for testing purposes.
- Strength recommendations and character suggestions for improvement.
- Clean GUI interface suitable for cybersecurity education and penetration testing demos.

---

## Installation
Clone the repository: 
```
git clone https://github.com/sujalkharatmol2732-crypto/Password-Strength-Checker.git
```
Navigate to the project directory:
```
cd Password-Strength-Checker
```
Run the application:  
```
python pass.py
```

**Requirements:** Python 3.6+ with Tkinter (included in standard Python installation). No additional dependencies needed.

---

## Usage
- Launch `pass.py` to open the GUI.
- Enter a password in the input field.
- Toggle **Show Password** to reveal input for testing.
- View the circular strength meter and detailed feedback.
- Use generated suggestions to create stronger passwords.

Perfect for cybersecurity students practicing password security assessment.
